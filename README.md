# 🤖 Telegram Bot

Умный Telegram-бот с поддержкой голосовых и текстовых сообщений, встроенной клавиатурой для ответов на часто задаваемые вопросы и автоматическим созданием базы данных при первом запуске.

---

## ✨ Возможности

- 🎙️ **Обработка аудиосообщений** — бот принимает голосовые сообщения, конвертирует аудио из `.ogg` в `.wav` через `pydub` и распознаёт речь с помощью `SpeechRecognition`. Результат обрабатывается как обычный текст.
- 💬 **Обработка текстовых сообщений** — бот анализирует входящие сообщения и формирует ответ. Данные пользователей и история диалога сохраняются в базе данных.
- ❓ **FAQ через ReplyKeyboardMarkup** — постоянная клавиатура с кнопками часто задаваемых вопросов. Пользователь одним нажатием выбирает нужный вопрос и мгновенно получает готовый ответ.
- 🗄️ **Автоматическая база данных** — база данных создаётся самостоятельно при первом запуске бота. Никаких ручных миграций не требуется.

---

## 📦 Установка зависимостей

```bash
pip install telebot SpeechRecognition pydub
```

Для работы `pydub` также необходимо установить системную утилиту **ffmpeg**:

| Платформа | Команда |
|-----------|---------|
| Windows | Скачать с [ffmpeg.org](https://ffmpeg.org) и добавить в папку с проектом|
| Linux | `sudo apt install ffmpeg` |
| macOS | `brew install ffmpeg` |

### Описание библиотек

| Библиотека | Назначение |
|------------|------------|
| `pyTelegramBotAPI` | Основа бота. Обёртка над Telegram Bot API — обработка сообщений, кнопки, вебхуки |
| `SpeechRecognition` | Распознавание речи. Поддерживает Google Speech API и офлайн-движки |
| `pydub` | Конвертация аудио. Преобразует голосовые `.ogg` от Telegram в `.wav` для распознавания |

## ⚙️ Конфигурация

Перед запуском откройте файл `config.py` и вставьте ваш API-токен бота:

```python
# Токен выдаёт @BotFather в Telegram при создании бота командой /newbot
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"

```

> ⚠️ **Никогда не публикуйте токен в открытом репозитории!** Добавьте `config.py` в `.gitignore` или используйте переменные окружения.

---

## 🚀 Запуск

```bash
python bot.py
```

При первом запуске база данных создастся автоматически.
